# Softmax-Regression

  - n[l] ( number of units in the output layer ) - number of classes to predict
  - Z[l] = W[l]*A[l-1] + b[l]
  - Activation Function:
    - t = exp(Z[l])
    - A[l] = exp(Z[l])/sum(t[i])
    - Ai[l] = ti/sum(ti)
